    url类型用于应该包含 URL 地址的输入域.
    在提交表单时,会自动验证 url 域的值.

    number类型用于应该包含数值的输入域.

表单元素

  • datalist
    datalist 元素规定输入域的选项列表.
    列表是通过 datalist 内的 option 元素创建的.
    如需把 datalist 绑定到输入域，请用输入域的 list 属性引用 datalist 的 id
    实例

  • keygen
    keygen 元素提供一种验证用户的可靠方法.
    keygen 元素是密钥对生成器.当表单提交时,会生成两个键,一个公钥,一个私钥.公钥发送到服务器,私钥存储到客户端.
    实例

表单属性

  • form 属性

    • autocomplete
      autocomplete 属性规定 form 或 input 域应该拥有的自动完成功能.
      注释：autocomplete 适用于

      标签，以及以下类型的 标签：text, search, url, telephone, email, password, datepickers, range 以及 color.
      当用户在自动完成域中开始输入时，浏览器应该在该域中显示填写的选项.
      实例

    • novalidate
      novalidate 属性规定在提交表单时不应该验证 form 或者 input 域.
      注释：novalidate 属性适用于

      以及以下类型的 标签：text, search, url, telephone, email, password, date pickers, range 以及 color.
      实例

  • input 属性

    • autocomplete

    • autofocus
      autofocus 属性规定页面加载时,域自动获取焦点.
      实例

    • form
      form 属性规定输入域所属的一个或多个表单.
      form 属性必须引用所属表单的 id.
      实例

    • multiple
      multiple 属性规定输入域可选择多个值.
      注释：multiple 属性适用于以下类型的 标签：email 和 file.
      实例

    • placeholder
      placeholder 属性提供一种提示,描述输入域所期待的值.
      注释：placeholder 属性适用于以下类型的 标签：text, search, url, telephone, email 以及 password.
      提示（hint）会在输入域为空时显示出现，会在输入域获得焦点时消失.
      实例

    • required
      required 属性规定必须在提交之前填写输入域.
      注释：required 属性适用于以下类型的 标签：text, search, url, telephone, email, password, date pickers, number, checkbox, radio 以及 file.
      实例

    • pattern
      pattern 属性规定用于验证 input 域的模式, 该模式是正则表达式.
      注释：pattern 属性适用于以下类型的 标签：text, search, url, telephone, email 以及 password.
      实例

    • min max step
      min、max 和 step 属性用于为包含数字或日期的 input 类型规定限定.
      max 属性规定输入域所允许的最大值.
      min 属性规定输入域所允许的最小值.
      step 属性为输入域规定合法的数字间隔（如果 step=”3”，则合法的数是 -3,0,3,6 等）.
      注释：min、max 和 step 属性适用于以下类型的 标签：date pickers、number 以及 range.
      实例

    • width 和 height
      height 和 width 属性规定用于 image 类型的 input 标签的图像高度和宽度.
      实例

    • 表单重写属性
      表单重写属性（form override attributes）允许您重写 form 元素的某些属性设定.
      formaction - 重写表单的 action 属性
      formenctype - 重写表单的 enctype 属性
      formmethod - 重写表单的 method 属性
      formnovalidate - 重写表单的 novalidate 属性
      formtarget - 重写表单的 target 属性
      注释：表单重写属性适用于以下类型的 标签：submit 和 image.
      实例

jQuery-总结

发表于 2014-01-15

jQuery 总结

对 jQuery 的认识

  • jQuery 是一个快速简洁的 JavaScript 代码库, 可以简化查询 DOM 对象, 处理事件, 制作动画, 处理 Ajax 交互过程
  • jQuery的优势
    • 轻量级,体积小,使用灵巧
    • 强大的选择器
    • 出色的 DOM 操作的封装
    • 出色的 浏览器兼容性
    • 可靠的事件处理机制
    • 完善的 Ajax
    • 链式编程,隐式迭代
    • 方便的选择页面元素
    • 动态更改页面样式
    • 控制响应事件
    • 易扩展,插件丰富

  • 快速使用

  • DOM 对象和 jQuery 对象的相互转换
    var $nav = $(nav) //原生转化成 jquery
    var nav = #nav.get(0) //jquery转化成原生

jQuery 选择器

  • 基本选择器
    • id选择器: $("#btn")
    • 类选择器: $(".class")
    • 标签选择器: $("span")
  • 层级选择器
    • 后代选择器: $("ul li")
    • 子代选择器: $("ul>li")
  • 过滤选择器
    • 选择匹配的第一个元素: $("li")
    • 选择索引为 index 的元素: $("li:eq(index)")
    • 选择索引为 奇数 的元素: $("li:odd")
    • 选择索引为 偶数 的元素: $("li:even")
    • 匹配找到的第一个元素: $("li:first")
    • 匹配找到的最后一个元素: $("li:last")
  • 筛选选择器
    • 在指定元素的所有后代中找某元素: $("ul").find("li")
    • 在指定元素的子代中找某元素: $("ul").children("li")
    • 在指定元素的同级元素中找某元素: $("ul").children("li").eq("2")
    • 在指定元素的所有兄弟元素中找某些元素: $("ul").children("li").eq("2").siblings("li")
    • 返回指定元素的父元素: $("li").parent("ul")
    • 返回指定元素的前一个指定元素: $("li").prev('li')
    • 返回指定元素的后一个指定元素: $("li").next('li')

属性和样式操作

  • 操作样式
    • 赋值: css("width",200)
    • 获取: css("width")
  • 操作类样式
    • 添加类: addClass("current")
    • 移除类: removeClass("current")
    • 切换类(有则删,无则加): toggleClass("current")
    • 判断有没有类: hasClass("current")
  • 操作属性
    • 操作一般属性
      attr():有参赋值,无参获取: $("input").attr("name","池子")
      prop():其属性值一般是 boolean值,多用在 checked,selected,disabled
      removeAttr():移除属性: $("input").removeAttr("name")
    • 操作内容属性
      text():有参赋值,无参获取,不识别标签: $("input").text("我爱北京天安门")
      html():有参赋值,无参获取,识别标签: $("input").html("我爱北京天安门")
      val():有参赋值,无参获取,标签中的 value 属性: $("input").val("我爱北京天安门")
  • 关于元素定位的操作
    • offset():目标在当前视窗的相对偏移,有参赋值,无参获取
      $("img").offset({left:100,top:100})
    • position():目标相对于最近的一个position()样式属性设置为relative或者absolute的祖父节点的相对偏移,有参赋值,无参获取
      $("img").position({left:100,top:100})
    • scrollTop():滚动条距离上方的距离,有参赋值,无参获取
      $("img").scrollTop(100)
    • scrollLeft():滚动条距离左侧的距离,有参赋值,无参获取
      $("img").scrollLeft(100)

jQuery DOM操作

  • 动态创建节点
    • html():动态创建并添加,有参赋值,无参获取,此方式会把之前的内容层叠掉
      ul.html("<li>你好吗</li>")
    • $():动态创建
      $("<li>你好吗</li>")
  • 添加节点
    • append():在调用者的所有子元素后面添加
      $("ul").append($("<li>你好吗</li>"))
    • prepend():在调用者的所有子元素前面添加
      $("ul").prepend($(<li>hello world</li>)
    • after():添加到调用者的后面
      $("li").after($(<li>hello world</li>)
    • before():添加到调用者的前面
      $("li").before($(<li>hello world</li>)
  • 删除节点
    • remove():删除该节点及所包含的后代节点
      $("ul").remove()
    • detach():与 remove 不同之处是所绑定的事件和数据都会保留下来
      $("ul").detach()
    • empty():删除该节点的所有后代节点
      $("ul").empty()
  • 复制节点
    • clone():深层复制该节点,可选参数 true,可以复制该节点的事件
      $("ul").clone()

jQuery 动画

  • 显示和隐藏 (改变宽高和不透明度)
    • show( speed,callback): 显示
      speed:表示动画运行多久(slow=0.6/normal=0.4/fast=0.2)
      callback:表示动画结束后立即执行的函数
    • hide( speed,callback): 隐藏
    • toggle( speed,callback): 切换
  • 滑入和滑出 (改变高度)
    • slideDown( speed,callback): 向下滑出
    • slideUp( speed,callback): 向上滑入
    • slideToggle( speed,callback): 切换
  • 淡入和淡出 (改变不透明度)
    • fideOut( speed,callback): 淡出
    • fideIn( speed,callback): 淡入
    • fadeToggle( speed,callback): 切换
    • fadeTo( speed, opacity, callback): 淡到什么程度
  • 自定义动画
    • animate(json, speed, callback)
      json: 要执行动画的 css 属性,必选
      speed: 执行动画时长,可选
      callback: 动画结束后立即执行的函数
  • 停止动画
    • stop(boolean1, boolean2)
      boolean1: true表示后续动画停止, false表示后续动画不停止
      boolear2: true表示当前动画立即执行完毕, false表示当前动画不执行完毕

jQuery 事件

  • 事件对象 event
    • event.data: 获取传递给事件处理程序的额外数据
    • event.type: 获取事件类型
    • event.target: 获取触发事件的元素,不一定 === this
    • event.currentTarget: 等同于 this
    • event.preventDefault(): 阻止事件默认行为
    • event.stopPropagation(): 阻止事件冒泡
    • event.keyCode: 针对于 keypress 事件,获取键盘键的数字
    • event.which: 针对于鼠标单击事件中,获取鼠标的左 中 右键
    • event.pageX / event.pageY: 获取光标相对于页面的 x 坐标和 y 坐标
    • event.clientX / event.clientY: 获取光标对于浏览器窗口的水平和垂直坐标
    • event.screenX / event.screenY: 获取光标对于电脑屏幕的水平和垂直坐标
  • 事件绑定
    • 直接绑定
      以 click 事件为例: $("div").click(callback)
    • bind():
      为每个匹配的元素绑定一个或多个的事件,只适用于当前的元素
      $(selector).bind(event, data(可选), callback)
      $("div").bind("click mouseenter", callback)
    • delegate():
      为指定元素添加一个或多个事件,适用于当前和未来的元素
      $(selector).delegate(childSelector, event, data(可选), callback)
      $("ul").delegate($("li"), "click mouseenter", {"name":"张三"}, callback)
    • on(): 当前常用
      为指定元素添加一个或多个事件,适用于当前和未来的元素
      $(selector).on(event, childSelector(可选), data(可选), callback)
      $("div").on("click mouseenter", callback)

jQuery Ajax

  • $.ajax(configObj)
    这是一个 jQuery对 ajax 封装的最基础步,通过这个函数可以完成异步通讯的所有功能.

  • $.post(configObj)
    这是对 $.ajax 的进一步封装,减少了参数,简化了操作,但是运用范围更小了.只能采用 post 方式提交,只能异步访问服务器,不能进行错误处理.

  • $.get(configObj)
    它和 $.post一样,这个函数是对 get 方法的提交数据进行封装,只能使用在 get 提交提交数据解决异步刷新的方式上.使用方式跟 $.post 一样

  • $.getJSON(configObj)
    这个是进一步的封装,也就是对返回数据类型为 json 进行操作
    他有三个参数: url, data, callback.
