音频 audio元素

  • audio 的使用
    实例

    audio 元素支持三种音频格式:Ogg Vorbis, MP3, Wav.每种格式对各个浏览器的兼容性不同.
    audio 元素允许多个 source 元素.
    source 元素可以链接不同的音频文件.
    浏览器将使用第一个可识别的格式.
    实例

  • audio 的属性
    | 属性 | 值 | 描述 |
    | —- | —- | —- |
    | autoplay | autoplay | 自动播放 |
    | controls | controls | 显示音频控件 |
    | loop | loop | 循环播放 |
    | preload | preload | 页面加载时进行加载,并预备播放,使用 autoplay 会忽视该属性 |
    | src | url | 音频的 url |
