jQuery 总结

对 jQuery 的认识

  • jQuery 是一个快速简洁的 JavaScript 代码库, 可以简化查询 DOM 对象, 处理事件, 制作动画, 处理 Ajax 交互过程
  • jQuery的优势
    • 轻量级,体积小,使用灵巧
    • 强大的选择器
    • 出色的 DOM 操作的封装
    • 出色的 浏览器兼容性
    • 可靠的事件处理机制
    • 完善的 Ajax
    • 链式编程,隐式迭代
    • 方便的选择页面元素
    • 动态更改页面样式
    • 控制响应事件
    • 易扩展,插件丰富

  • 快速使用

  • DOM 对象和 jQuery 对象的相互转换
    var $nav = $(nav) //原生转化成 jquery
    var nav = #nav.get(0) //jquery转化成原生

jQuery 选择器

  • 基本选择器
    • id选择器: $("#btn")
    • 类选择器: $(".class")
    • 标签选择器: $("span")
  • 层级选择器
    • 后代选择器: $("ul li")
    • 子代选择器: $("ul>li")
  • 过滤选择器
    • 选择匹配的第一个元素: $("li")
    • 选择索引为 index 的元素: $("li:eq(index)")
    • 选择索引为 奇数 的元素: $("li:odd")
    • 选择索引为 偶数 的元素: $("li:even")
    • 匹配找到的第一个元素: $("li:first")
    • 匹配找到的最后一个元素: $("li:last")
  • 筛选选择器
    • 在指定元素的所有后代中找某元素: $("ul").find("li")
    • 在指定元素的子代中找某元素: $("ul").children("li")
    • 在指定元素的同级元素中找某元素: $("ul").children("li").eq("2")
    • 在指定元素的所有兄弟元素中找某些元素: $("ul").children("li").eq("2").siblings("li")
    • 返回指定元素的父元素: $("li").parent("ul")
    • 返回指定元素的前一个指定元素: $("li").prev('li')
    • 返回指定元素的后一个指定元素: $("li").next('li')

属性和样式操作

  • 操作样式
    • 赋值: css("width",200)
    • 获取: css("width")
  • 操作类样式
    • 添加类: addClass("current")
    • 移除类: removeClass("current")
    • 切换类(有则删,无则加): toggleClass("current")
    • 判断有没有类: hasClass("current")
  • 操作属性
    • 操作一般属性
      attr():有参赋值,无参获取: $("input").attr("name","池子")
      prop():其属性值一般是 boolean值,多用在 checked,selected,disabled
      removeAttr():移除属性: $("input").removeAttr("name")
    • 操作内容属性
      text():有参赋值,无参获取,不识别标签: $("input").text("我爱北京天安门")
      html():有参赋值,无参获取,识别标签: $("input").html("我爱北京天安门")
      val():有参赋值,无参获取,标签中的 value 属性: $("input").val("我爱北京天安门")
  • 关于元素定位的操作
    • offset():目标在当前视窗的相对偏移,有参赋值,无参获取
      $("img").offset({left:100,top:100})
    • position():目标相对于最近的一个position()样式属性设置为relative或者absolute的祖父节点的相对偏移,有参赋值,无参获取
      $("img").position({left:100,top:100})
    • scrollTop():滚动条距离上方的距离,有参赋值,无参获取
      $("img").scrollTop(100)
    • scrollLeft():滚动条距离左侧的距离,有参赋值,无参获取
      $("img").scrollLeft(100)

jQuery DOM操作

  • 动态创建节点
    • html():动态创建并添加,有参赋值,无参获取,此方式会把之前的内容层叠掉
      ul.html("<li>你好吗</li>")
    • $():动态创建
      $("<li>你好吗</li>")
  • 添加节点
    • append():在调用者的所有子元素后面添加
      $("ul").append($("<li>你好吗</li>"))
    • prepend():在调用者的所有子元素前面添加
      $("ul").prepend($(<li>hello world</li>)
    • after():添加到调用者的后面
      $("li").after($(<li>hello world</li>)
    • before():添加到调用者的前面
      $("li").before($(<li>hello world</li>)
  • 删除节点
    • remove():删除该节点及所包含的后代节点
      $("ul").remove()
    • detach():与 remove 不同之处是所绑定的事件和数据都会保留下来
      $("ul").detach()
    • empty():删除该节点的所有后代节点
      $("ul").empty()
  • 复制节点
    • clone():深层复制该节点,可选参数 true,可以复制该节点的事件
      $("ul").clone()
